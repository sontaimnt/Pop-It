# Pop-It
A Pop It game written in tkinter

# Features:-

- Light Weight
- Cross Platform/Architecture(works on x64 , i686)
- Beautiful/Minimal UI
- Made with X11 Colors(no external colors)
- 3x3 , 4x4 , 5x5 Game Mode
